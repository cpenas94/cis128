Progressive Web Applications

(PWA)

About PWA's:

Progressive Web App (PWA) is a type of web application that uses modern web technologies to provide an app-like experience to users. PWAs are designed to be reliable, fast, and engaging, and can be accessed through a web browser, without the need to download and install them from an app store.

PWAs are built using web technologies such as HTML, CSS and Javascript which can be designed to work offline with data being stored in the browser's cache. They also can be installed on a user's home screen, making them easily accessible, providing an enjoyable user experience.

It is designed to be responsive and work across multiple devices and screen sizes. It has a layout that adapts to the screen size and the orientation of the devices. It can also take advantage of device features such as push notifications, camera access, geolocation and providing an engaging user experience.
